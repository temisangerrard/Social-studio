# UGC Tab Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Create a dedicated top-level UGC workspace that uses brand context but is fully isolated from the Studio slideshow/editorial workflow and routing state.

**Architecture:** Add a new `UGC` view with dedicated frontend state and request builders, add dedicated backend UGC draft/generate endpoints, and remove UGC entry points from the Studio style workflow. Reuse shared brand/storage primitives where safe, but do not reuse the Studio route-preview/style-selection path for UGC generation.

**Tech Stack:** Vanilla JS frontend modules, existing Node/TS backend, HTML/CSS views, Node test runner with `tsx`
